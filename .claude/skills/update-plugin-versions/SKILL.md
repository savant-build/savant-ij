---
name: update-plugin-versions
description: Updates the latest version of the plugins in the docs. Use when the user asks to "update plugin versions".
---

When updating the plugin versions in the docs, follow these steps:

1. Determine the version of each plugin by finding the most recent tag in Git for each repository
2. Find the plugins documentation page and update the `LATEST VERSION` information
3. Update the code snippet in the `Loading the plugin` section to use the latest version of the plugin
